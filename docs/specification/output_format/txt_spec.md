# 流水线txt(纯文本）格式数据输出标准

## 原则

1. 每个规范点有且仅有一种正确实现
2. 着重定义txt中如何表达表格、列表、代码、数学公式，
3. 正确区分格式特定的符号和正文符号

## 适用场景

1. 从HTML、PDF等提取的数据，给到训练模型时需要以txt格式。
2. 从公开数据集得到的数据，有的是md格式的，需要从md格式转为txt格式。

> ⚠️有从txt格式得到的原始数据，不适用于本规则。
