# 流水线Markdown格式数据输出标准

## 原则

1. 使用markdown的最小子集满足业务需求
2. 最大可能兼容开源markdown库，方便网页、各类现有编辑器渲染
3. 每个规范点有且仅有一种正确实现
4. 着重定义markdown中格式特定的符号与正文符号的转换关系
