    Particular Solution

    40xy' -ln(x^8) = 0

    Initial Condition: y(1)=31

    Work:
    40x (dy/dx) - ln(x^8) = 0

    40x(dy/dx) = ln (x^8)

    dy/dx = ln(x^8) / 40x

    Sdy = Sln(x^8) / 40x

    And then...stuck (if its even right thus far)
